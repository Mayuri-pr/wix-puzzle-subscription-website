## 📌 Project Overview
This project is a fully responsive subscription-based website developed using Wix from scratch without using any pre-built templates. The website is designed for a puzzle subscription service, focusing on a clean layout, engaging UI, and smooth user experience.

---

## 🚀 Features
- Fully responsive subscription-based website  
- Custom-built layout without using templates  
- Clean and modern UI/UX  
- Structured content flow for better user engagement  
- User-friendly navigation  
- SEO optimized structure
